# Using an ESP32
 - Neo-M8N (Same as nR52 link below)
 - Any ESP32 dev board
 - (Optional) any 5v battery solution for the ESP32, GPS module is powered from 3.3V
## Wiring up the ESP32
|ESP32| GPS Module |
|--|--|
| GND | GND |
| 3V3 | VCC |
| GPIO17 | TX |
| GPIO16 | RX |
